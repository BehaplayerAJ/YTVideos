Сценарий для Shorts: "Скрытые фишки ArrayList, о которых вы не знали!"

[Интро — 3 секунды]

    [На экране быстро появляется заголовок]: "Скрытые фишки ArrayList!"

    Текст: "Ты думаешь, что знаешь всё об ArrayList? Давай я удивлю тебя!"

[Основная часть — 30 секунд]

    **Фишка 1: **``

        "Когда ты знаешь, сколько элементов будет добавлено, ArrayList можно оптимизировать! Используй метод ensureCapacity(), чтобы избежать лишних операций увеличения размера."

        [На экране]:
        ArrayList<Integer> list = new ArrayList<>();
        list.ensureCapacity(1000); // Ускоряет добавление

        "Эта простая строка может серьёзно ускорить код, если ты работаешь с большими данными."

    **Фишка 2: **``

        "Хочешь работать только с частью списка? Используй subList()!"

        [На экране]:
        ArrayList<String> list = new ArrayList<>(List.of("A", "B", "C", "D"));
        List<String> subList = list.subList(1, 3); // ["B", "C"]

        "Но будь осторожен! Изменения в subList отразятся на оригинальном списке."

    Фишка 3: Быстрое удаление всех элементов

        "Очистка больших списков может быть долгой. Но вот трюк:"

        [На экране]:
        list.clear();
        // или
        list = new ArrayList<>(); // Быстрее, если старая ссылка не нужна

        "Создание нового списка иногда быстрее, чем вызов clear()!"

[Заключение — 7 секунд]

    "Теперь ты знаешь больше об ArrayList! Какие из этих трюков ты будешь использовать? Напиши в комментариях!"

    [На экране текст]: "Ставь лайк и подписывайся, если было полезно!"

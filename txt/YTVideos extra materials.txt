Да, есть несколько инструментов и редакторов, которые помогут сделать код на Java визуально привлекательным для твоих видео. Вот несколько рекомендаций:
1. Инструменты для записи красивого кода
a) Carbon

    Сайт: https://carbon.now.sh/
    Преимущества:
        Создание красивых изображений кода.
        Выбор шрифта, темы и фона (включая градиенты и прозрачность).
        Настройка размеров и теней.
    Как использовать:
        Скопируй свой Java-код.
        Вставь его на сайт.
        Настрой стиль и тему под твой вкус.
        Скачай изображение для вставки в ролик.

b) Codeimg

    Сайт: https://codeimg.io/
    Преимущества:
        Простота использования.
        Возможность добавить заголовки и логотипы.
        Поддержка темных и светлых тем.

2. Редакторы кода с красивыми темами
a) Visual Studio Code (VS Code)

    Установи расширение Prettier для форматирования.
    Используй красивые темы, такие как:
        One Dark Pro (темная).
        Material Theme (с вариациями).
        Monokai Pro.
    Для записи экрана можешь включить "Zen Mode" (Ctrl+K Z), чтобы убрать отвлекающие элементы.

b) IntelliJ IDEA

    Очень популярная IDE для Java с поддержкой тем.
    Красивые темы:
        Dracula (по умолчанию).
        Material Theme (доступно через плагин).
        Gruvbox (нестандартная).
    Включи Presentation Mode (View → Appearance → Enter Presentation Mode), чтобы код смотрелся крупнее и минималистично.

3. Обработка видео

    Если ты записываешь код, попробуй OBS Studio для записи экрана. Ты можешь настроить обрезку окна и добавить наложение.
    Добавь анимацию, подсветку или лупы, чтобы акцентировать внимание на конкретных строках кода.
